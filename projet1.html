Enquête sur les pratiques culturelles

Contexte

Ce projet s'inscrit dans le cadre d'un travail universitaire visant à analyser les habitudes culturelles des étudiants.

Objectif

L'objectif était de collecter des données fiables et de comprendre les pratiques culturelles à partir d'une enquête.

Données utilisées

Les données proviennent d'un questionnaire administré via LimeSurvey auprès d'étudiants.

Méthodologie

  • J'ai conçu le questionnaire.
  • J'ai collecté les réponses.
  • J'ai nettoyé les données sous Excel.
  • J'ai structuré les variables pour garantir la cohérence statistique.
  • J'ai réalisé une analyse descriptive.

Résultats

L'analyse a permis d'identifier les principales tendances des pratiques culturelles. Les résultats ont été synthétisés dans un rapport clair et structuré.
